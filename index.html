Como fazer dual boot com Windows e Linux

Se você deseja fazer um dual boot deve se ater a alguns detalhes importantes na hora da instalação, nesse tutorial vamos ver como particionar o HD/SSD, as partições para Windows e Linux, além de uma partição de arquivos, que não é obrigatória, porém é algo útil se você deseja compartilhar seus arquivos entre os dois Sistemas Operacionais.



O que é um dual boot?

Um dual boot basicamente consiste em colocar dois sistemas operacionais diferentes no mesmo HD/SSD.


Para que ele serve?

O uso pode variar de acordo com a necessidade do usuário, alguns podem fazer porque necessitam de aplicativos específicos que somente podem ser baixados naquele determinado sistema, alguns porque tem diferentes usos para cada SO (Sistema Operacional), por exemplo: Windows para jogos e Linux para programação e alguns pelo simples fato de que querem se aventurar em fazer um dual boot.


O que é necessário para fazer um dual boot?

Primeiramente você precisará de uma mídia bootável, seja ela um pendrive, um CD, um HD/SSD externo, nesse link você encontra como fazer uma mídia bootável.
